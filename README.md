# 生成模拟图片

## 使用地址

url: http://node.sunshine1990.com/image-mocker

## 参数

- width: 图片宽度,默认 200
- height: 图片高度,默认 200
- fColor: 文字颜色,默认 #eee
- bColor: 图片背景色,默认 #000

## 事例

- http://node.sunshine1990.com/image-mocker?width=400&height=300&fColor=red&bColor=blue
